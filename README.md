## Instalación

Clona el repositorio en terminal mediante los siguientes comandos:

```bash
git clone https://github.com/dixnne/OUILookup.git
cd OUILookup
chmod +x OUILookup.py
```

## Uso

```bash
python3 OUILookup.py [-a,-m {aa:bb:cc...}]
```
